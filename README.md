# Week 7 - Operator Overloading

## Overview

This week introduces operator overloading to extend the functionality of user-defined data types.

## Learning Objectives

* Understand operator overloading
* Customize operator behavior
* Improve code readability
* Implement mathematical operations on objects

## Experiments

### Experiment 1: Complex Number Class

Overload operators for:

* Addition (+)
* Assignment (=)
* Multiplication (*)

### Experiment 2: Month Class

Implement:

* Prefix Increment (++)
* Postfix Increment (++)
* Prefix Decrement (--)
* Postfix Decrement (--)
* Stream Operators (<< and >>)

## Concepts Covered

* Operator Overloading
* Complex Numbers
* Stream Operators
* Prefix and Postfix Operations

## Language Used

C++

## Outcome

Students learned how to create intuitive and powerful custom data types using operator overloading.
